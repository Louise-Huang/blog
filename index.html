jQuery入門

什麼是jQuery？

jQuery是一個以Javascript來編寫的函式庫，讓開發者能夠更方便操作網頁DOM元素、建立動畫效果以及使用Ajax串接API。

安裝jQuery

CDN

可以從Google提供的開發者函式庫中尋找

Download API

直接搜尋CDN的網址並右鍵存儲至本地端

環境建置

通常為了節省效能，會將導入的部分放在body標籤裡面最下方，避免JQuery導入完成但HTML DOM架構尚未讀取完成所浪費的等待時間。

VS Code插件JQuery Code Snippets，輸入「jqdo」可以快速設置環境

選擇器與事件

jQuery以$符號加上class與id的名稱或標籤名稱作為DOM元素的選擇器，搭配滑鼠事件製造出動畫效果。

Click()事件、show()、hide()、toggle()開關效果

最常見的是以滑鼠點擊事件來執行動畫效果，toogle()可以自動切換show/hide

API介紹

API實作流程

  1. 取得資料
  2. 將資料轉換成HTML放到畫面上

什麼是API?

應用程式介面 Application Programming Interface
舉例：有一台飲料販賣機，你想喝一瓶奶茶，按了飲料機上的奶茶按鈕，奶茶掉下來，取出奶茶。
奶茶就是你想取得的資料，按鈕就是API，透過API發送需求來取得資料就是串接API的流程。
