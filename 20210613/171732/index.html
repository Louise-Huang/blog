發表於 2021-06-13 分類於 Front-end technology

Chart.js介紹

Chart.js是一款很好用的Javascript套件，可以用來製作八種常見的圖表，甚至可以將圖表種類混合顯示，並且以小動畫的方式呈現，極具美感！
由於目前做的專案需要顯示圖表，因而認識Chart.js這個套件，這邊就順便做個學習筆記，主要記錄我有使用到的部分，其他的部分可以參考官網喲！

Chart.js主要設定

以我寫的程式碼為例，主要是設定以下四個部分，或許有更多設定，可以參考官網！

  1. type 圖表類型：line, bar, radar, doughnut, polarArea, bubble, scatter
  2. labels 圖表X軸標籤
  3. datasets 資料集
  4. options 圖表設定

我個人習慣先將labels和datasets的內容另外處理好再代進去config，避免整個config物件被拉得很長很長。

目前我有使用到的options設定有幾個

  • plugins 圖表外觀設定：包含圖例(legend)位置、標題的位置/顏色/字型大小/padding、提示框(tooltip)內容等等
  • elements 元素設定：包含線條彎曲度(lineTension)、線條到原點是否填滿顏色(fill)、點的大小
  • scales X軸Y軸座標標籤設定：包含Y軸的最小值與最大值、Y軸間距(stepSize)、使用callback函式讓Y軸標籤字串加上單位’%’

最後來個大總結吧，將所有設定好的合併起來！

結尾

可以依照自己的喜好與需求設定更多樣式，滑鼠移動到該筆資料時，會以動畫顯示資料提示框喲！當然，動畫也是可以自行設定的！Chart.js是不是很好用呢？大家都來玩玩看吧！
